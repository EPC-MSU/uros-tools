Скрипт, который создает апдейт из диска с файловой системой Байкала и файла загрузочного раздела.
Для работы скрипту нужно примерно 4 GB свободного места в /tmp. Выходной файл апдейта создается в текущей директории.
Скрипт ожидает наличие файлов check_home.sh, tango.conf в директории запуска.
Требуется наличие dd, mount, umount, zerofree, cpio, fsck на системе запуска скрипта.
Версия питона: 3.5 или выше.

Установка зависимостей: pip3 install -r requirements.txt

Запуск: python3 snapshoter.py --partition копируемый_ssd_раздел --boot файл_загрузочного_раздела --semversion версия_апдейта --name имя_апдейта
Опции можно также сокращать до "-p", "-b", "-s", "-n" соответственно.

Пример запуска: python3 snapshoter.py --partition /dev/sdb1 --boot bootpart.img --semversion 1.0.0 --name newupdate
